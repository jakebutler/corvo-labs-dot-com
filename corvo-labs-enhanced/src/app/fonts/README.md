# Cabinet Grotesk Font Files

To complete the font setup, please download Cabinet Grotesk font files and place them in this directory:

Required files:
- `CabinetGrotesk-Regular.woff2` (weight: 400)
- `CabinetGrotesk-Medium.woff2` (weight: 500)
- `CabinetGrotesk-Bold.woff2` (weight: 700)
- `CabinetGrotesk-Extrabold.woff2` (weight: 800)
- `CabinetGrotesk-Black.woff2` (weight: 900)

You can download Cabinet Grotesk from:
- https://www.fontshare.com/fonts/cabinet-grotesk
- Or purchase from the official source

Once the font files are in place, the font will automatically load via `next/font/local` in `fonts.ts`.
