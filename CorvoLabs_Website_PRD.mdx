## 9. Deployment Guide

### Prerequisites
- GitHub repo: `jakebutler/corvo-labs-dot-com`
- Node.js 20+ and npm or pnpm installed

### Steps
1. **Clone Repo:**
   ```bash
   git clone https://github.com/jakebutler/corvo-labs-dot-com.git
   cd corvo-labs-dot-com
   ```
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Run Locally:**
   ```bash
   npm run dev
   ```
4. **Deploy to Vercel:**
   - Push to GitHub main branch
   - Link repository on [vercel.com](https://vercel.com)
   - Set build command: `next build`
   - Output directory: `.next`
5. **Verify:**
   - Test metadata and blog rendering
   - Confirm giscus + newsletter forms work
   - Generate sitemap via `next-sitemap`
